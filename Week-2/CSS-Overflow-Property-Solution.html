The overflow Property

The overflow property specifies whether to clip content or to add scrollbars when an element's content is too big to fit in a specified area.

overflow: scroll;

With the value set to scroll the overflow is clipped, and a scrollbar is added to see the rest of the content. With this value, the space for scroll bar will always appear.

overflow: hidden;

With the hidden value, the overflow is clipped, and the rest of the content outside of the box will be invisible. This can make an image a circle. Notice how the text is cut of when it reaches the bottom of the box.

overflow: hidden; On an Image

overflow: auto;

Auto is similar to scroll, except with the value set to auto, the scroll bars only appear when needed, if the content exceeds the bounds of the box.

overflow: visible; (default)

This is the default value for the overflow property. The overflow is not clipped, so the content renders outside the element's box. Notice that the text and image extend below the box boundaries.
